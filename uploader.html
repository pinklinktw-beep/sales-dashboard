📊 銷售資料更新工具

上傳當月 Excel → 自動轉換 → 下載 JSON → 上傳到 GitHub

📋 使用步驟

1
選擇當月 Excel 檔案（可以同時選多個月份）
檔名需包含月份，例如 2026_06月.xlsx
2
等待轉換完成後，點「下載 biz_data.json」
3
打開 GitHub repo → data/ 資料夾 → 點 biz_data.json → 右上角鉛筆 → 貼上新內容 → Commit
4
約 1 分鐘後 儀表板 自動更新

📂 上傳 Excel 檔案

支援同時上傳多個月份（01月～12月）。2026_XX月.xlsx 格式

📂
點擊選擇或拖曳 Excel 至此
支援多選，格式：2026_06月.xlsx

⚠️ 注意事項

• 這個工具只處理經銷體系的資料
• 大使體系目前仍只有 2026-04 的真實資料
• 排除名單（930530 何佩蓉）已固定在程式中
• 轉換過程全部在你的瀏覽器進行，不會上傳到任何伺服器
